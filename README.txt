INTRODUCTION
------------

Admin Toolbar intends to improve the default Drupal Layout Builder management, providing a fast and full access to all blocks & paragraph components being used in Layout builder.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/layout_builder_usage_reports

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/search/layout_builder_usage_reports


FEATURES
--------

 * Lists all nodes using layout builder,
 * Blocks & Block types being used in the layout,
 * Paragraph components & Paragraph types being used in the layout,
 * Options to filter by node bundle, language, block type, paragraph type...
